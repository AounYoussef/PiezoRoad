# 🚗⚡ PiezoRoad

An interactive React simulation that demonstrates how piezoelectric roads can harvest the mechanical pressure generated by vehicles and convert it into renewable electrical energy.

## 🌍 Overview

PiezoRoad is an educational simulation designed to visualize the concept of piezoelectric energy harvesting in road infrastructure. As cars travel over piezoelectric sensors embedded beneath the road surface, the simulation generates electricity in real time, illustrating how traffic can become a source of clean energy.

This project aims to raise awareness of sustainable technologies and smart city innovations through an engaging and interactive web experience.

## ✨ Features

- 🚗 Animated vehicle traffic simulation
- ⚡ Real-time energy generation
- 📈 Live electricity production counter
- 🎮 Interactive controls
- 🌱 Educational demonstration of piezoelectric technology

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/AounYoussef/PiezoRoad.git
```

Navigate into the project:

```bash
cd PiezoRoad
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

or, if you're using Create React App:

```bash
npm start
```

Open your browser and visit:

```
http://localhost:5173
```

*(or `http://localhost:3000` if using Create React App).*

## 💡 How It Works

1. Vehicles move across the simulated road.
2. Piezoelectric sensors detect the pressure applied by each vehicle.
3. Mechanical energy is converted into electrical energy.
4. The generated energy is accumulated and displayed in real time.

The simulation represents the concept visually and is intended for educational purposes.
