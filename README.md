## 🌍 Overview

**PiezoRoad** is an interactive web simulation built to explore the concept of harvesting renewable energy from road traffic using **piezoelectric sensors**.

Every vehicle traveling across a road applies mechanical pressure to the surface. Instead of allowing this energy to dissipate, piezoelectric materials can convert part of that mechanical stress into usable electrical energy.

This project visualizes that concept through a responsive simulation where users can observe vehicles generating electricity in real time while monitoring the condition of individual sensors embedded beneath the road.

## 💡 Why Piezoelectric Roads?

Roads experience constant pressure from thousands of vehicles every day.

Piezoelectric technology offers an innovative way to recover a portion of this otherwise wasted mechanical energy by converting it into electricity.

Although the energy produced by each vehicle is relatively small, large-scale installations could contribute to powering applications such as:

- Smart street lighting
- Traffic management systems
- Environmental monitoring sensors
- Public infrastructure
- Low-power smart city devices

PiezoRoad demonstrates this concept through an interactive educational experience.

---

## ⚙️ How It Works

1. Vehicles travel across the simulated roadway.
2. Piezoelectric sensors detect the pressure generated by each vehicle.
3. Mechanical energy is converted into electrical energy.
4. Generated electricity is accumulated and displayed in real time.
5. Users can inspect individual sensors to monitor their condition and damage level.

> **Note**
>
> PiezoRoad is an educational simulation inspired by real-world piezoelectric energy harvesting research. It is designed to visualize the concept rather than accurately model the electrical output of a physical system.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/AounYoussef/PiezoRoad.git
cd PiezoRoad
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```
