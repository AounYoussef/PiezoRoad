🚗⚡ PiezoRoad

Transforming traffic into clean energy through the power of piezoelectric technology.

PiezoRoad is an interactive React and TypeScript simulation that demonstrates how piezoelectric sensors embedded beneath road surfaces can convert the mechanical pressure generated by moving vehicles into renewable electrical energy.

Rather than simply explaining the concept, PiezoRoad allows users to visualize energy generation in real time as vehicles travel across a smart road, providing an engaging way to explore one of the emerging technologies in sustainable infrastructure.

🌍 Overview

Modern roads experience millions of vehicle passes every day, producing a tremendous amount of mechanical pressure that is typically wasted.

PiezoRoad explores how this untapped energy can be harvested using piezoelectric sensors placed beneath the road surface. Every passing vehicle applies force to the sensors, which convert mechanical stress into electrical energy that could potentially power applications such as:

💡 Street lighting
🚦 Traffic signals
📡 Smart city sensors
🔋 Battery storage systems
🏙️ Public infrastructure

This project is an educational simulation inspired by real-world research into renewable energy harvesting and smart transportation systems.

🧠 How It Works

The simulation follows four simple steps:

Vehicles move across the smart road.
Piezoelectric sensors detect the pressure applied by each vehicle.
Mechanical energy is converted into electrical energy.
The generated energy is accumulated and displayed in real time.

Additionally, the simulation models the condition of each individual sensor, allowing users to inspect sensor health and better understand how infrastructure maintenance can affect system performance.

🚀 Getting Started
Clone the repository

cd PiezoRoad
Install dependencies
npm install
Start the development server
npm run dev

Open your browser and visit:

http://localhost:5173
Build for production
npm run build
Preview the production build
npm run preview
